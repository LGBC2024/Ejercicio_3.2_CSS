## VINCULACION HTML

1. Hoja de estilo externa usando etiqueta <link></link>

```html
<link rel="stylesheet" href="./styles.css" />
```

2. Usando la etiqueta "style"

```html
<style>
    h1 {
        color: red;
    }
</style>
```

3. De manera inline en el elemento con el atributo "style"

```html
<h1 style="color: red">CSS</h1>
```
